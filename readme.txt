== Description ==

[Nachrichten] Mit dem Nachrichten.plus-Plugin können Sie aktuelle Nachrichten auf Ihren Wordpress-Blog mühelos einfügen und über unser Partnerprogramm zusätzlich mit Premium-Nachrichten Geld verdienen. Ganz einfach den Shortcode in Ihre Seite oder Artikel einbinden und schon erhalten Sie für ihre Besucher die letzten Nachrichten. Ein echter Mehrwert für z.B. Blogger und Influencer. Unser Partnerprogramm für Premium-Nachrichten bietet Ihnen die Möglichkeit, mit unseren Inhalten zusätzliches Geld zu verdienen. Als Influencer, Facebook Gruppen Admin, Telegram Kanal Admin, Webseiten- oder Blogbetreiber erhalten Sie eine Beteiligung über 3 Ebenen ( Multi Level Marketing ) Ebene 1 = 25%, Ebene 2 = 7%, Ebene 3 = 3% für den Abschluss eines Premium-Pakets. (Zur Anmeldung geht es hier entlang: <a href="https://nachrichten.plus/partnerprogramm/" target="_blank">https://nachrichten.plus/partnerprogramm/</a> )

= Current features =
Sie können auf Hunderte Artikel zugreifen und mithilfe von WordPress-Widgets oder Shortcodes von Ihnen angepasste Nachrichten auf Ihrer Website anzeigen.

* Legen Sie alle Nachrichten oder Premium Nachrichten fest
* Legen Sie die Anzahl der Artikel fest
* Abstracts anzeigen
* Datum anzeigen
* Premium-Inhalte
* Partner-ID: Verdienen Sie Geld mit Premium Nachrichten

== Installation ==

1. **Installieren:** Installieren Sie die Nachrichten entweder über das WordPress.org-Plugin-Verzeichnis oder indem Sie die Dateien auf Ihren Server hochladen.
2. **Newsfeeds erstellen:** Erstellen Sie Ihren Nachrichten-feed, indem Sie auf der Registerkarte „Shortcode generieren“ einen Shortcode generieren. Fügen Sie diesen Shortcode in Beiträge oder Seiten ein, auf denen Sie Nachrichten anzeigen möchten, ODER erstellen Sie Ihre Nachrichten über „Darstellung“ > „Widgets“. Ziehen Sie aus dem Widgets-Panel das Widget „Nachrichten“ in die gewünschte Seitenleiste oder den gewünschten Widget-Bereich, wo Sie Ihren Newsfeed anzeigen möchten. Bearbeiten Sie die Widget-Funktionen, um Ihren Newsfeed zu erstellen/bearbeiten.
